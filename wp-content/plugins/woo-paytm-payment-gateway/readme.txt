== Description ==
This plugin is used to integrate Paytm payment gateway with Woocommerce.

Pay Your Order with Paytm is the simplest of the available ways to do payment integration to your website. 

The person who clicks the Pay via paytm button will be taken to Pay with Paytm website where user will complete the transaction via Paytm and Pay with Paytm returns back the browser handle to the website after payment confirmation. Website will also receive a notification, which will be a server-to-server call as soon as the payment is done.

== Installation ==
Installation

Login your Wordpress admin and navigate to Plugins->Add New 

Find the WooCommerce Paytm Pay plugin and activate the module.

After the successful activation of module, Visit the WooCommerce => settings page, and click on the Checkout tab.

Click On Checkout and display "Pay With Paytm" payment gateway.

Enable the Payment Method, and Set all details and  Click Save.(See screenshot)

Now you can see *pay with paytm* payment option on checkout page.

For Create Merchant ID Go to <a href="https://business.paytm.com/payments" target="_blank">Paytm Website</a>.

* Signup/Sign In on paywithpaytm.com and fill in the profile details and bank details.
